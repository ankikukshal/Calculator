🧮 HTML CSS JavaScript Calculator

This project is a simple and interactive **Calculator Web Application** built using HTML, CSS, and JavaScript. It allows users to perform basic arithmetic operations such as addition, subtraction, multiplication, division, and percentage calculations. The calculator features a clean and modern user interface with responsive button layouts and a display screen for showing input and results.

Users can input numbers and operators using on-screen buttons, clear the display, delete the last entered value, and compute results instantly. The JavaScript logic handles user input, updates the display dynamically, and evaluates expressions using built-in functions.

This project is ideal for beginners to understand core web development concepts such as DOM manipulation, event handling, styling with CSS Grid, and basic JavaScript functions. It can be further improved by adding keyboard support, advanced operations, and better error handling.

---

🚀 Features

* Basic arithmetic operations (+, −, ×, ÷, %)
* Clear (C) and backspace (⌫) functionality
* Responsive grid layout
* Error handling for invalid expressions
* Modern UI design

---

🛠️ Technologies Used

* HTML5
* CSS3 (Grid & Styling)
* JavaScript (DOM Manipulation & Logic)

---

▶️ How to Run

1. Save the file as `.html`
2. Open it in any web browser
3. Start performing calculations
